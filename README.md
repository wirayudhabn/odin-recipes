# Odin Recipes 🍳

A foundational HTML project from [The Odin Project](https://www.theodinproject.com/) curriculum. 

## 🎯 Project Overview
This project is a simple recipe website built entirely with raw HTML. The main goal of this project is to practice the fundamentals of web development structure before diving into CSS styling. 

It serves as a digital recipe book, linking a main index page to several distinct recipe pages.

## 🛠️ Skills Demonstrated
Through building this project, I successfully applied and solidified my understanding of:
* **HTML Boilerplate:** Setting up the correct structure (`<!DOCTYPE html>`, `<head>`, `<body>`, etc.).
* **Text Formatting:** Using headings (`<h1>` to `<h6>`), paragraphs (`<p>`), and emphasizing text (`<strong>`, `<em>`).
* **Lists:** Implementing ordered (`<ol>`) and unordered (`<ul>`) lists for ingredients and instructions.
* **Hyperlinks:** Creating absolute links to external websites and relative links to navigate between local HTML files.
* **Images:** Embedding images and using the `alt` attribute for accessibility.
* **Version Control:** Practicing the standard Git workflow (`add`, `commit`, `push`) using the command line.
